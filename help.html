Frequently Asked Questions

What is PhishGuard and how does it work?

PhishGuard is a browser extension that protects you from phishing attacks by checking websites you visit against databases of known phishing sites and analyzing webpage content for suspicious elements.

When you visit a website, PhishGuard works in the background to:

  • Check the URL against PhishTank, Google Safe Browsing, and our custom database of Nigeria-specific phishing sites
  • Scan the webpage content for phishing indicators like suspicious forms, brand impersonation, and hidden elements
  • Show a warning if a potential phishing attempt is detected
Does PhishGuard slow down my browsing?

PhishGuard is designed to be lightweight and efficient. The extension performs most checks in the background and uses optimized algorithms to minimize any impact on your browsing experience.

In most cases, you won't notice any slowdown. URL checks are performed quickly, and content scanning happens after the page has loaded so it doesn't delay page rendering.

What information does PhishGuard collect about my browsing?

PhishGuard respects your privacy. The extension checks URLs against phishing databases but does not store a history of the websites you visit on our servers.

Content scanning is performed locally in your browser, and we do not transmit or store the content of websites you visit.

If enabled in settings, we collect anonymous usage statistics like the number of phishing attempts blocked to help improve the extension.

For complete details, please see our Privacy Policy.

Do I need API keys for PhishGuard to work?

PhishGuard can work without API keys, but adding them enhances the protection:

  • Without API keys: PhishGuard will use basic URL analysis and content scanning to detect phishing attempts
  • With API keys: PhishGuard can check URLs against PhishTank, Google Safe Browsing, and our custom database for more comprehensive protection

You can add API keys in the extension settings. Instructions for obtaining free API keys are provided on the settings page.

What should I do if I encounter a phishing site?

If PhishGuard detects a phishing site, it will show a warning with options to go back to safety or proceed (not recommended). If you encounter a phishing site that wasn't detected:

  1. Do not enter any personal information on the site
  2. Click the PhishGuard icon in your browser toolbar
  3. Click "Report This Site" to submit the URL for analysis
  4. Leave the site immediately

Your report helps improve protection for all users.

Can PhishGuard detect all phishing attempts?

While PhishGuard uses advanced techniques to detect phishing attempts, no solution can guarantee 100% protection. Phishing attacks are constantly evolving, and new threats emerge regularly.

PhishGuard provides strong protection against known phishing sites and common phishing techniques, but it's still important to stay vigilant and follow good security practices:

  • Be cautious about clicking links in emails or messages
  • Verify website URLs before entering sensitive information
  • Look for HTTPS (padlock icon) in the address bar
  • Be suspicious of urgent requests for personal information

Troubleshooting

PhishGuard icon is grayed out or not responding

This usually indicates that the extension is not properly initialized or has encountered an error.

Solution:
  1. Try refreshing the page
  2. Restart your browser
  3. Disable and re-enable the extension:
    • Go to chrome://extensions/
    • Find PhishGuard and toggle it off, then on again
  4. If the problem persists, try reinstalling the extension

PhishGuard is showing false positives (marking safe sites as phishing)

Sometimes legitimate websites may be incorrectly flagged as phishing sites due to certain characteristics they share with malicious sites.

Solution:
  1. Adjust your warning level in settings to "Low" to reduce false positives
  2. Report the false positive:
    • Click the PhishGuard icon
    • Click "This is a false positive" in the popup
  3. Temporarily disable specific checks in settings that might be causing the issue

API key errors or connection issues

If you've added API keys but are seeing errors related to them, there might be issues with the keys or the connection to the API services.

Solution:
  1. Verify that your API keys are entered correctly in the settings
  2. Check if the API services are operational (visit their status pages)
  3. Ensure your internet connection is stable
  4. Try using the extension without API keys temporarily to see if the issue persists

Extension is slowing down browsing

If you notice performance issues while using PhishGuard, you can adjust settings to improve performance.

Solution:
  1. Disable content scanning in settings (this will still allow URL checking)
  2. Reduce the number of phishing databases checked (e.g., disable one or more of PhishTank, Google Safe Browsing, or custom database)
  3. Set warning level to "Low" to reduce the intensity of checks
  4. Make sure your browser and system are up to date

Settings are not saving

If your settings aren't being saved when you change them, there might be issues with browser storage.

Solution:
  1. Clear browser cache and cookies
  2. Make sure you have sufficient disk space
  3. Check if other extensions are working properly (to rule out browser-wide issues)
  4. Try using the "Reset to Defaults" button in settings, then reconfigure your preferences

How to Use PhishGuard

Getting Started

  1. Installation: After installing PhishGuard from the Chrome Web Store, you'll see the PhishGuard icon in your browser toolbar.
  2. Initial Setup: Click the icon and select "Settings" to configure your preferences and add API keys if desired.
  3. Basic Protection: PhishGuard works automatically in the background, checking websites as you browse.

Understanding the Icon Colors

The PhishGuard icon in your browser toolbar changes color to indicate the status of the current website:

  • Blue: PhishGuard is active and no threats are detected
  • Yellow: Potential suspicious elements detected (low confidence)
  • Red: Phishing site detected (high confidence)
  • Gray: PhishGuard is disabled or not active on this page

Responding to Warnings

When PhishGuard detects a potential phishing site, you'll see a warning overlay with the following options:

  • Back to Safety: Returns you to the previous page (recommended)
  • Proceed Anyway: Continues to the site despite the warning (use with caution)

The warning will include details about why the site was flagged, helping you make an informed decision.

Reporting Phishing Sites

  1. Click the PhishGuard icon in your browser toolbar
  2. Click "Report This Site" in the popup
  3. Fill out the report form with details about the suspected phishing site
  4. Click "Submit Report"

Your reports help improve protection for all users and contribute to the fight against phishing.

Customizing Settings

To access settings, click the PhishGuard icon and select "Settings". You can customize:

  • Protection Settings: Enable/disable different phishing databases and content scanning
  • Warning Level: Choose between Low, Medium, or High sensitivity
  • API Keys: Add keys for enhanced protection
  • Data & Privacy: Control what data is collected and shared

Viewing Statistics

The PhishGuard popup shows statistics about your protection:

  • Number of sites checked
  • Number of threats blocked
  • Days the extension has been active

These statistics help you understand how PhishGuard is protecting you over time.
