Getting Started

Configure Your Settings

Click on the PhishGuard icon in your browser toolbar and select "Settings" to customize your protection level and preferences.

Add Your API Keys (Optional)

For enhanced protection, you can add API keys for PhishTank and Google Safe Browsing in the settings page.

Browse with Confidence

PhishGuard works automatically in the background. If a suspicious site is detected, you'll see a warning before any harm is done.

Report Suspicious Sites

Help improve protection for everyone by reporting suspected phishing sites through the extension.

How to Identify Phishing Attempts

While PhishGuard works to protect you automatically, it's still important to know how to spot phishing attempts yourself:

Common Signs of Phishing:

  • Suspicious URLs that mimic legitimate websites (e.g., "faceb00k.com" instead of "facebook.com")
  • Poor grammar and spelling errors
  • Requests for personal information or login credentials
  • Urgent or threatening language to create pressure
  • Offers that seem too good to be true

Security Tip:

Always check the URL in your browser's address bar before entering any sensitive information. Make sure it uses HTTPS (look for the padlock icon) and matches the official domain of the service you're using.
